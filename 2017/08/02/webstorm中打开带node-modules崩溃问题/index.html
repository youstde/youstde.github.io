webstorm中打开带node_modules崩溃的问题

1.点击菜单栏中webstorm，选择preferences => languages&frameworks => JavaScript => Libraries

然后将node_modules去除掉

2.在左边菜单栏中右击node_modules然后选择 Mark Directory As 选择 Excluded
